Para correr el algoritmo utilizado en el primer punto>
1. Adaptar las variables ruta y ruta2 segun direccion del proyecto
2. Correr el archivo punto1.py 

Este archivo tiene las variables verLOCAL y ver LASER que pueden ser evaluadas:
verLocal=true, para guardar el plot local de las mediciones del sensor, global evaluado en false.
verLASER=true, para visualizar los rayos del laser que va desde el sensor hasta la medicion, o no visualizarlo en false.

Las figuras se guardan en la carpeta de results  scripts.
